# 🎥 H-Scan

A modern surveillance dashboard UI for monitoring CCTV feeds, alerts, and security insights through a clean and scalable interface.

> 🚧 **Status:** Frontend completed. Backend integration in progress.

## ✨ Features

- 📹 CCTV-style monitoring dashboard
- 🚨 Security alert panel
- 📊 Dashboard analytics cards
- 🧭 Sidebar navigation
- 🧩 Reusable React components
- 📱 Responsive interface

## 🛠️ Tech Stack

- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Vite
- Lucide React

## 🚀 Run Locally

```bash
npm install
npm run dev
```

## 🌱 Future Improvements

- Real-time alert streaming
- Backend API integration
- Authentication & role-based access
- Historical feed management
